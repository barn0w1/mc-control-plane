# Checkpoints

Checkpointは、単なる作業期間ではなく、独立して意味のあるsubsystemまたは能力の完成点です。
各checkpoint directoryには、目標、architecture、現在の実装状況、未決事項をまとめます。

## Active

- [Host Control System v1](host-control-v1/README.md)

## Planned after the active checkpoint

Host Control System v1の完了後に、Workload、Data、Minecraft Serverなどの上位layerを別checkpointとして定義します。
現時点では、それらの詳細なresource modelや実装計画を固定しません。
